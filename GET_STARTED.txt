📋 NEXT STEPS (Takes 5 minutes):

Step 1: Get Google Maps API Key
   → Visit: https://console.cloud.google.com/
   → Enable "Maps JavaScript API"
   → Create API Key
   
Step 2: Configure Environment
   $ cp .env.example .env.local
   
   Edit .env.local and add:
   VITE_GOOGLE_MAPS_API_KEY=your_api_key_here
   
Step 3: Run the Application
   $ npm install
   $ npm run dev
   
   Open: http://localhost:5173

🐛 TROUBLESHOOTING:

   Map not loading?
   → Check if VITE_GOOGLE_MAPS_API_KEY is set
   → Enable billing in Google Cloud (includes $200/month free)
   → Check browser console (F12) for errors
   
   No buildings showing?
   → Verify public/data/gombe_open_buildings.geojson exists
   → Run: python scripts/fetch_open_buildings.py
   

💡 TIPS:

   • Google Maps requires billing enabled (free tier available)
   • Zoom level 14+ for best satellite imagery
   • Click buildings to see property details
   • Use filters to search by type, size, or confidence
