🔑 Step 1: Get Mapbox Token
   → Go to: https://account.mapbox.com/access-tokens/
   → Copy your default public token (starts with pk.)

🚀 Step 2: Add Token & Deploy

   METHOD A - Command Line (30 seconds):
   ```
   npx vercel env add VITE_MAPBOX_TOKEN production
   [paste your token]
   npx vercel --prod
   ```

   METHOD B - Vercel Dashboard (1 minute):
   1. Visit: https://vercel.com/mmtukur/fastfind360/settings/environment-variables
   2. Click "Add New"
   3. Key: VITE_MAPBOX_TOKEN
   4. Value: [your token]
   5. Environment: Production
   6. Save & Redeploy
